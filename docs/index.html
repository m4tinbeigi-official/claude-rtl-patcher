01 / ویژگی‌ها

یک تجربه ساده، از نصب تا اجرا.

Aa

تایپوگرافی فارسی

فونت Vazirmatn برای خوانایی بهتر در گفتگوها و رابط کاربری Claude.

↔

RTL هوشمند

نسخه Claude را تشخیص می‌دهد و فقط در صورت نیاز patch کامل RTL را اعمال می‌کند.

⌘

امن و قابل بازگشت

قبل از تغییر backup می‌سازد، integrity macOS را به‌روزرسانی می‌کند و restore دارد.

02 / نصب

یک دستور. تمام مراحل.

از نسخه ۱.۱.۲ فایل‌های اجرایی مستقل منتشر می‌شوند؛ Node.js لازم نیست.

curl -fsSL https://raw.githubusercontent.com/m4tinbeigi-official/claude-rtl-patcher/main/install.sh | bash
irm https://raw.githubusercontent.com/m4tinbeigi-official/claude-rtl-patcher/main/install.ps1 | iex

نصب‌کننده مسیر Claude را تشخیص می‌دهد، backup می‌گیرد و منوی patch را اجرا می‌کند.
